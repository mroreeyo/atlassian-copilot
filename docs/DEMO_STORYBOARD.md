# DEMO_STORYBOARD.md

## 1. Demo Goal

Show that this is not “just Jira search.”

It is a safe AI Copilot layer over Jira/Confluence with:

- dark-first chat UI
- MCP tool plan
- Broker-streamed fictional MCP-shaped evidence for P0
- Broker-streamed AI/mock summary with the same SSE contract
- source citations
- action approval UX

## 2. Demo Script

1. Open Copilot.
2. Show calm dark-first chat UI.
3. Ask:

```txt
NFS 프로젝트에서 이번 주 완료되지 않은 High 이슈를 찾아서 요약해줘.
관련 Confluence 문서도 같이 보고, 조치 댓글 초안까지 만들어줘.
```

4. Compact Tool Plan appears.
5. Progress Strip updates.
6. Evidence Card shows Jira and Confluence sources.
7. AI Summary streams through the Broker SSE contract.
8. Feedback buttons are visible.
9. Action Review Card appears for Jira comment.
10. Explain that write action is not executed until approval.
11. Show History entry for the run.
12. Show Settings connection status.

## 3. What to say in interview

> Jira/Confluence는 이미 데이터를 저장하고 검색할 수 있습니다. 이 프로젝트는 그것을 대체하는 것이 아니라, 사용자가 자연어로 업무 맥락을 요청하면 Broker가 MCP-shaped 근거를 안전한 계약으로 전달하고, Broker-streamed 요약과 조치 초안을 생성하며, 변경 작업은 Action Review에서 승인 후에도 P0에서는 실행되지 않도록 설계한 AI UX layer입니다. 실제 MCP/OpenAI 연결은 Broker-only 경계 안에서 P1/P1.5로 확장합니다.
